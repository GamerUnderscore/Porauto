
# Port Autonome

Ceci est un projet scolaire, en cours de développement.
Port autonome est un panel de gestion permettant de controller une grue.

Ce projet est développé en html (+JS, ...) puis est compilé avec ElectronJS pour en faire un éxécutable.

# Important:
Cette application est faite pour fonctionner uniquement avec le bon materiel. L'application est donc faite sur-mesure.

# Dépendances:
- NodeJS
- ElectronJS
- Bootstap
- Handstonable
- JQuery
- GSAP
- Icônes Google
